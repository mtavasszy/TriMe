# TriMe
A project to reproduce an image as accurately as possible using semi-transparent colored triangles. 
The code uses a hill climbing algorithm using C++ and OpenGL3 to estimate the best configuration and print this out to the screen.

## Dependencies
- ImGui
- OpenGL3

## Inspired by
- Evolution of Mona Lisa by Roger Johansonn (https://rogerjohansson.blog/2008/12/07/genetic-programming-evolution-of-mona-lisa/) 
- A javascript implementation of the algorithm by AlteredQualia (https://alteredqualia.com/visualization/evolve/)
